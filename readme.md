## 🚀 Como Executar os Projetos

Como estes são projetos estáticos, você não precisa instalar nada. Basta seguir um dos métodos abaixo:

### 1. Execução Direta (Mais simples)
1.  Faça o download ou clone este repositório.
2.  Navegue até a pasta do projeto que deseja visualizar.
3.  Encontre o arquivo `index.html`.
4.  **Dê um duplo clique** no arquivo ou arraste-o para dentro do seu navegador (Chrome, Firefox, Edge).

### 2. Usando o VS Code (Recomendado)
Se você é desenvolvedor e usa o VS Code, recomendo a extensão **Live Server**:
1.  Abra a pasta do projeto no VS Code.
2.  Clique com o botão direito sobre o arquivo `index.html`.
3.  Selecione a opção **"Open with Live Server"**.
4.  O projeto abrirá automaticamente no seu navegador com atualização em tempo real.
