# 🎨 Cards de serviços

Um componente de grade de serviços moderno e responsivo, construído com HTML e CSS puro. O projeto apresenta cards elegantes com efeitos de interatividade e ícones personalizados.

---

## 🚀 Demonstração

O projeto consiste em uma seção de serviços que se adapta automaticamente a diferentes tamanhos de tela, utilizando um layout escuro (*Dark Mode*) com detalhes em azul neon.

### Principais Funcionalidades:
* **Layout Responsivo:** Os cards se ajustam dinamicamente usando `flex-wrap` e Media Queries.
* **Efeitos de Hover:** Transições suaves de escala e mudança de cor na borda para melhorar a experiência do usuário.
* **Tipografia Moderna:** Integração com a fonte *Roboto* via Google Fonts.
* **Ícones:** Utilização da biblioteca *Boxicons* para representação visual dos serviços.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica dos cards.
* **CSS3:** * Flexbox para alinhamento e distribuição.
    * Design Responsivo (Mobile First/Adaptativo).
* **Boxicons:** Biblioteca de ícones vetoriais.

---

## 📂 Como visualizar o projeto

Como este é um projeto de frontend estático, você pode executá-lo de forma muito simples:

1. Faça o download dos arquivos `index.html` e `style.css`.
2. Certifique-se de que ambos estão na mesma pasta.
3. Abra o arquivo `index.html` em qualquer navegador (Chrome, Edge, Firefox, etc).

> **Dica:** Se estiver usando o VS Code, clique com o botão direito no `index.html` e selecione **"Open with Live Server"** para ver as alterações em tempo real.

---

## 📈 Metas de Aprendizado (Nível Iniciante)

Este projeto foi fundamental para praticar:
* **Box Model:** Entendimento de padding, border e margin.
* **Flexbox:** Uso de `flex: 1 1 304px` para criar cards que crescem e encolhem conforme o espaço disponível.
* **Pseudo-classes:** Uso do `:hover` para criar interatividade sem JavaScript.
